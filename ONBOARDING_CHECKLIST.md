# Checklist de Onboarding - Novos Clientes ERP BateriaExpert

## Antes do Primeiro Acesso

### Configuração Técnica

- [ ] Criar tenant no banco central
- [ ] Provisionar banco de dados no Supabase
- [ ] Configurar subdomínio (`cliente.erp.com`)
- [ ] Configurar plano de assinatura escolhido
- [ ] Enviar credenciais de acesso ao administrador do cliente

### Configuração do Tenant

- [ ] Personalizar white label (logo, cores)
- [ ] Cadastrar informações da empresa (CNPJ, endereço)
- [ ] Configurar e-mails de notificação

## Primeiro Acesso

### Super Admin do Cliente (Dono)

- [ ] Acessar o sistema com credenciais temporárias
- [ ] Redefinir senha
- [ ] Criar outros usuários (gestor, vendedores, técnicos)
- [ ] Cadastrar produtos (marcas, modelos, baterias)
- [ ] Cadastrar veículos (fabricantes, modelos)
- [ ] Configurar CFOPs e impostos (se necessário)

### Importação de Dados (opcional)

- [ ] Importar clientes de sistema legado
- [ ] Importar estoque inicial
- [ ] Importar histórico de vendas (últimos 12 meses)

- [ ] Clientes reais cadastrados
- [ ] Estoque real conferido

- [ ] Agendamento de revisão mensal

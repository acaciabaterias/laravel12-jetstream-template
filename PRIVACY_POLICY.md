# Política de Privacidade - ERP BateriaExpert

## Dados Coletados

- Dados cadastrais: Nome, CNPJ/CPF, endereço, contato
- Dados de uso: Transações, vendas, estoque
- Dados técnicos: IP, logs de acesso

## Base Legal

- Execução de contrato (vendas, cobranças)
- Legítimo interesse (melhorias do sistema)
- Consentimento (marketing, comunicações)

## Uso dos Dados

- Operacionalizar vendas, estoque e entregas
- Emitir notas fiscais
- Melhorar o serviço
- Cumprir obrigações legais

## Compartilhamento

- Não vendemos dados pessoais
- Compartilhamos com:
  - Parceiros de pagamento (Stripe/ASAAS)
  - Autoridades fiscais (notas fiscais)
  - Entregadores (endereços para entrega)

## Armazenamento e Segurança

- Dados criptografados em repouso
- Isolamento por tenant (banco de dados separado)
- Backup diário com retenção de 30 dias

## Direitos do Titular (LGPD)

- Acessar seus dados
- Corrigir dados incompletos
- Solicitar exclusão (quando aplicável)
- Portabilidade dos dados
- Revogar consentimento

## Retenção de Dados

- Dados fiscais: 5 anos (por lei)
- Dados cadastrais: enquanto ativo + 30 dias
- Histórico de vendas: 5 anos
